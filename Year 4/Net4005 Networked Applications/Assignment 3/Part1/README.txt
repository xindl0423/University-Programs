# Assignment 3 NET4005

**Description

Sending secure application messages between Client-Sever

**Getting Started

***Executing program

To run the server file use:
            -python server.py and this will run the server section of the file
            -sample format:
                        -'python server.py'
            -The sever will generate RSA keys if there is none and start listening for messages
To run the client file use:
            -python client.py
            -It generated RSA keys if there is none.
            -It will prompt the user for input which it then encrypts and signs
